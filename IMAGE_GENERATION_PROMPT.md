# Image Generation Prompt for Team Photos

## Image Specifications for Pro Motion Physiotherapy Website

Use this prompt when generating or creating team member photos to ensure they fit properly in the website cards.

### Recommended Image Specifications:

**Dimensions:**
- **Aspect Ratio:** 3:4 (portrait orientation)
- **Recommended Size:** 1200px width × 1600px height (or 900px × 1200px minimum)
- **File Format:** PNG or JPG
- **File Size:** Under 500KB (optimized for web)

### Image Generation Prompt:

```
Create a professional headshot photograph of a healthcare professional (physiotherapist or occupational therapist). 

Technical Requirements:
- Portrait orientation with 3:4 aspect ratio (width:height)
- Dimensions: 1200px × 1600px
- High resolution, sharp focus on the subject
- Professional lighting with soft, even illumination
- Clean, neutral background (light gray or white) that can be easily replaced
- Subject should be centered and take up approximately 60-70% of the frame height
- Leave some space above the head (headroom) and below the shoulders

Composition Guidelines:
- Subject should be positioned in the center of the frame
- Head and shoulders visible, with some upper torso
- Subject looking directly at camera with friendly, professional expression
- Professional attire (business casual or medical scrubs)
- Good contrast between subject and background
- No distracting elements in the background

Style:
- Professional, warm, and approachable
- Medical/healthcare professional aesthetic
- Clean and modern
- Suitable for a physiotherapy clinic website
```

### Alternative Detailed Prompt for AI Image Generators:

```
Professional headshot portrait of a healthcare professional, 3:4 vertical aspect ratio, 1200x1600 pixels, high resolution, sharp focus, professional studio lighting, neutral light gray background, subject centered in frame, head and shoulders visible, friendly professional expression, business casual attire, clean composition, medical professional aesthetic, suitable for healthcare website, portrait photography style, professional quality
```

### Key Points for Best Fit:

1. **Aspect Ratio:** Always use 3:4 (portrait) - this matches the card layout
2. **Subject Position:** Center the person vertically with some space above and below
3. **Background:** Use a neutral gray background that matches the website's gray-200 color (#E5E7EB)
4. **Subject Size:** Person should fill about 60-70% of the image height
5. **Consistency:** All team photos should follow the same composition style for visual consistency

### Color Reference:
- Website gray background: `#E5E7EB` (gray-200)
- Use this exact color for the background to match seamlessly

### File Naming:
- Use descriptive names: `team-member-name.png`
- Example: `ben-bastin-jose.png`, `lijo-paul.png`, `bency-thekkineth.png`

### Post-Processing Tips:
1. Ensure images are optimized for web (compressed but maintain quality)
2. All images should have the same dimensions for consistency
3. Crop to exact 3:4 ratio if needed
4. Adjust background color to match website gray if necessary
