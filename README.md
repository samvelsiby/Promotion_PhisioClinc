# Pro Motion Physiotherapy

A professional website for a physiotherapy clinic in Winnipeg, Manitoba.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

- `app/` - Next.js App Router pages and layouts
- `components/` - Reusable React components
- `public/` - Static assets (images, logos, etc.)

## Adding the Logo

Please add the logo PNG file to the `public/` directory as `logo.png`. The navbar component will automatically use it.

## Features

- **Responsive Navigation Bar**: Fully responsive navigation with mobile menu
- **Modern Design**: Clean, professional design matching clinic branding
- **TypeScript**: Type-safe development
